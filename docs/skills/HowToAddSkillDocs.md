# How To Add Skill Docs

Use this when adding a new project guide under `docs/skills`.

## Add A New Guide

1. Create one focused markdown file under `docs/skills`.
2. Name it as an action-oriented question, such as `HowToAddInputActions.md`.
3. Keep the file short and procedural.
4. Link it from `docs/skills/README.md`.
5. Link it from `AGENTS.md` if future agents should discover it early.

## Writing Rules

- Put durable project knowledge here, not temporary task notes.
- Prefer commands, file paths, invariants, and examples over broad explanation.
- Avoid duplicating another guide. Link to the sibling guide instead.
- Update an existing guide when a workflow changes.

## Good Candidates

- How to add input actions.
- How to add sprite animations.
- How to add level data.
- How to add renderer tests.
- How to add release signing.
